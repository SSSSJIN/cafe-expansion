# cafe-expansion

# cafe-expansion 프로젝트는 카카오웹페이지를 부분 clone 하는 프로젝트 입니다.

# 개발기간 : 06.06 ~ 06.27(약 3주)

## 주요 기능
- 기능1 : ’지금 핫한’, ‘실시간 랭킹＇
- 기능2 : 로고 클릭시 홈 화면 복귀
- 기능3 : 작품 상세정보 제공
- 기능4 : 에피소드를 통한 뷰어 기능
